## How to deploy
- [ ] At first , clone this repo
```bash
git clone https://github.com/shakil1819/BLRM-Solidity.git
```
- [ ] Install truffle,npm,ganache-cli,metamask wallet 
```bash
$ truffle compile
$ truffle migrate
$ truffle build
$ truffle dashboard
$ ganache #to check transactions, account in blockchain

```
